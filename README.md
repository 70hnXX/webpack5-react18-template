### 需要全局安装的插件

- npm install -g eslint
- npm i serve -g
- npm i commitizen -g

### 需要安装的 vscode 插件

- EditorConfig for VS Code
- Prettier - Code formatter
- ESLint

### 打包后启动服务

```
npm i serve -g
serve -s dist
```
